# 412. Fizz Buzz

Difficulty: Easy
Topics: string, math

## Problem

Return the array fizz/buzz/fizzbuzz/n for 1..n where multiples of 3 become Fizz, multiples of 5 Buzz, both FizzBuzz.

## Approach

Loop 1 to n, append the right string based on divisibility.

Time O(n), space O(n) for the output.

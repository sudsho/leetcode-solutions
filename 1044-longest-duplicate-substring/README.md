# 1044. Longest Duplicate Substring

Difficulty: Hard
Topics  : binary search, rolling hash

## Problem

Longest duplicated substring in s.

## Approach

Binary search on length; rolling hash to test if any duplicate of that length exists.

## Complexity

Time O(N log N) avg, space O(N).

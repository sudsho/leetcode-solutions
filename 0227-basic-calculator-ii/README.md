# 227. Basic Calculator II

Difficulty: Medium
Topics  : stack, string

## Problem

Evaluate string expression with +, -, *, /, integers; no parens.

## Approach

Stack of terms. * and / fold immediately with previous number.

## Complexity

Time O(N), space O(N).

# 572. Subtree of Another Tree

Difficulty: Easy
Topics  : tree, dfs, hash function

## Problem

Return True if subRoot is a subtree of root (matches some node's whole subtree).

## Approach

Recurse. At each node test if structurally equal to subRoot.

## Complexity

Time O(m*n), space O(h).

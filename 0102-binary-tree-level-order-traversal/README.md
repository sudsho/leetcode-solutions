# 102. Binary Tree Level Order Traversal

Difficulty: Medium
Topics  : tree, bfs, binary tree

## Problem

Given the root of a binary tree, return the level order traversal of its node values.

## Approach

BFS with a queue, taking one level at a time.

## Complexity

Time O(n), space O(n).

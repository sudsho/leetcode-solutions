# 347. Top K Frequent Elements

Difficulty: Medium
Topics  : array, hash table, heap, bucket sort

## Problem

Given an int array and integer k return the k most frequent elements.

## Approach

Count then bucket-sort by frequency. Or heap of size k.

## Complexity

Time O(n), space O(n).

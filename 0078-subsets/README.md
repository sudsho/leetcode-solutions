# 78. Subsets

Difficulty: Medium
Topics  : array, backtracking, bit manipulation

## Problem

Given a list of unique ints return all possible subsets (the power set).

## Approach

Backtracking. At each index either include or skip.

## Complexity

Time O(2^n * n), space O(n).

# 1011. Capacity To Ship Packages Within D Days

Difficulty: Medium
Topics  : binary search, greedy

## Problem

Min weight capacity to ship all packages within d days in given order.

## Approach

Binary search on capacity. Greedy simulate days needed.

## Complexity

Time O(n log sum), space O(1).

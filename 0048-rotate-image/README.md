# 48. Rotate Image

Difficulty: Medium
Topics  : array, math, matrix

## Problem

Rotate an n x n 2D matrix 90 degrees clockwise in place.

## Approach

Transpose then reverse each row. Or do the layer-by-layer four-way swap.

## Complexity

Time O(n^2), space O(1).

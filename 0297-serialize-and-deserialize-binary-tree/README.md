# 297. Serialize and Deserialize Binary Tree

Difficulty: Hard
Topics  : tree, design, dfs, bfs

## Problem

Design serialize and deserialize for a binary tree.

## Approach

Preorder with explicit None markers. Encode as a comma-joined string.

## Complexity

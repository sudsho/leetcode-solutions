# 344. Reverse String

Difficulty: Easy
Topics: string, two pointers

## Problem

Reverse a string in place. Input is a list of characters, must be modified in place with O(1) extra memory.

## Approach

Two pointers from each end, swap and walk inward.

## Complexity

Time O(n), space O(1).
